üî¢ Sudoku

Sudoku is a classic number puzzle game where you fill a 9x9 grid with numbers 1-9.

How to Play:

1. Click on any empty cell to select it

2. Type a number from 1-9

3. Fill all empty cells following these rules:

  • Each row must contain numbers 1-9 without repetition
  • Each column must contain numbers 1-9 without repetition
  • Each 3x3 box must contain numbers 1-9 without repetition

Features:

Timer: Tracks how long you've been playing
Pause: Pause the game and hide the grid
Check: Validate your solution (green = correct, red = incorrect)
New Game: Reset the puzzle and timer
Auto-save: Your progress is automatically saved

üéÆ Tetris

Tetris is a classic block-stacking game where you arrange falling pieces to clear lines.

How to Play:

1. Click "Start Game" to begin

2. Use keyboard controls to move and rotate falling pieces

3. Complete horizontal lines to clear them and score points

4. Game ends when pieces stack to the top

Controls:

‚Üê Move piece left
‚Üí Move piece right
‚Üì Move piece down faster
‚Üë Rotate piece

Features:

Score: Earn points by clearing lines
Level: Speed increases as you score more points
Lines: Track total lines cleared
Pause: Pause/resume the game anytime

Tips:

  • Try to keep the stack low and even
  • Clear multiple lines at once for bonus points
  • Plan ahead - look at the piece shape and color
  • Use the pause button if you need a break

üêç Snake

Snake is a classic arcade game where you control a growing snake that must eat food while avoiding obstacles.

How to Play:

1. Click "Start Game" to begin

2. Use arrow keys to control the snake's direction

3. Eat the red food to grow longer and score points

4. Avoid hitting the walls or your own tail

Controls:

‚Üë Move up
‚Üì Move down
‚Üê Move left
‚Üí Move right

Features:

Score: Earn 10 points for each food eaten
High Score: Your best score is saved locally
Pause: Pause/resume the game anytime

Tips:

  • Plan your path ahead to avoid trapping yourself
  • Use the edges carefully - they can be helpful or deadly
  • The snake moves continuously, so stay alert
  • You can't reverse direction (e.g., can't go left if moving right)
